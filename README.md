# ⚡ BayarKita - Aplikasi Pembayaran Tagihan Multi-Layanan

Aplikasi web frontend yang mensimulasikan proses pembayaran berbagai tagihan rutin masyarakat Indonesia serta pengisian pulsa/paket data. Aplikasi ini sepenuhnya berjalan di sisi klien (client-side) tanpa backend server nyata.

## 📋 Deskripsi

BayarKita adalah aplikasi pembayaran tagihan multi-layanan yang terinspirasi dari aplikasi seperti Tokopedia, DANA, dan PLN Mobile. Aplikasi ini dibangun untuk memenuhi tugas Pemrograman Web II dengan fokus pada:

- Perancangan antarmuka pengguna (UI) yang intuitif, modern, dan responsif
- Pengelolaan state dan interaktivitas menggunakan Vanilla JavaScript
- Simulasi alur pembayaran yang realistis
- Penyimpanan data menggunakan LocalStorage
- Pengalaman pengguna (UX) yang baik dengan loading state, validasi form, notifikasi, konfirmasi, dan bukti transaksi
